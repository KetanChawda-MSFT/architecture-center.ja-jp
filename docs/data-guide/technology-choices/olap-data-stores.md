<span data-ttu-id="d49cf-111">SQL Server Analysis Services (SSAS) は、ビジネス インテリジェンス アプリケーション向けに OLAP およびデータ マイニング機能を提供しています。</span><span class="sxs-lookup"><span data-stu-id="d49cf-111">SQL Server Analysis Services (SSAS) offers OLAP and data mining functionality for business intelligence applications.</span></span> <span data-ttu-id="d49cf-112">ローカル サーバーに SSAS をインストールするか、Azure の仮想マシン内でホストすることができます。</span><span class="sxs-lookup"><span data-stu-id="d49cf-112">You can either install SSAS on local servers, or host within a virtual machine in Azure.</span></span> <span data-ttu-id="d49cf-113">Azure Analysis Services は、SSAS と同じ主要機能を提供する完全管理のサービスです。</span><span class="sxs-lookup"><span data-stu-id="d49cf-113">Azure Analysis Services is a fully managed service that provides the same major features as SSAS.</span></span> <span data-ttu-id="d49cf-114">Azure Analysis Services では、組織のクラウドおよびオンプレミスにある[多様なデータ ソース](/azure/analysis-services/analysis-services-datasource)への接続がサポートされます。</span><span class="sxs-lookup"><span data-stu-id="d49cf-114">Azure Analysis Services supports connecting to [various data sources](/azure/analysis-services/analysis-services-datasource) in the cloud and on-premises in your organization.</span></span>
